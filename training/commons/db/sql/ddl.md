# Schema Creation

- CREATE SCHEMA [Name] [ [authorization] Auth ] {SchemaElementDefinition}

- A schema:
	- Contains: Domains, Tables, Indexes, Assertions, Views, Privileges.
	- Has a Name and Owner.
